# Data Analytics Project - SQL + Power BI

## 📌 Overview
This project demonstrates an end-to-end Data Analytics workflow using SQL and Power BI. It focuses on solving business problems through SQL queries and presenting insights through an interactive dashboard.

## 🚀 Features
- Data cleaning and preparation
- SQL analysis using:
  - Joins
  - Subqueries
  - Window Functions
  - Aggregate Functions
- Interactive Power BI Dashboard
- KPI Tracking
- Trend Analysis
- Visual Reports with Filters

## 🛠 Tools & Technologies
- SQL
- Power BI
- Data Visualization
- Data Analysis

## 📊 Dashboard Highlights
- Overview Summary
- Performance Metrics
- Trend Analysis
- Category Insights
- Interactive Filters

## 📂 Project Files
- `dataset.csv` - Raw dataset
- `queries.sql` – SQL questions and solutions
- `dashboard.pbix` – Power BI dashboard file

## 🎯 Purpose
This project was created to strengthen practical skills in SQL, dashboard creation, and business data analysis.
